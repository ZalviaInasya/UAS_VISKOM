# UAS_VISKOM

Project ini merupakan tugas UAS Mata Kuliah Visi Komputer yang berfokus pada deteksi emosi wajah secara real-time menggunakan kamera. Sistem dibangun dengan Python, OpenCV, dan Convolutional Neural Network (CNN), di mana model dilatih menggunakan dataset ekspresi wajah (angry, happy, sad, dll.), lalu diterapkan untuk mendeteksi emosi pengguna secara langsung melalui webcam. Project ini bertujuan untuk menerapkan konsep Computer Vision dari preprocessing data, pelatihan model, hingga implementasi real-time detection.
